# Installation — GaussianEditor C4D

Voir setup_windows.bat pour l'installation automatique.

## Installation manuelle

### 1. Prérequis Windows
- Windows 10/11
- WSL2 Ubuntu 22/24
- NVIDIA GPU (4GB VRAM minimum)
- Cinema 4D 2025+

### 2. Installation automatique (WSL2)

```bash
git clone https://github.com/stamboulii/GaussianEditor_C4D
cd GaussianEditor_C4D
chmod +x install.sh
./install.sh
```

### 3. Utilisation quotidienne

1. Double-clic sur `Demarrer_GaussianEditor.bat` (Bureau Windows)
2. Ouvrir Cinema 4D
3. Plugin → Cliquer **Demarrer**
4. Sélectionner une image → **Generer**
